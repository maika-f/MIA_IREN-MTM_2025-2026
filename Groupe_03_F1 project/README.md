# F1 AI Betting Predictions

## Project Overview

An AI-powered Formula 1 betting prediction system that fetches historical F1 news and race data, analyzes it using AI agents, and generates betting predictions with odds similar to prediction markets.

## Objectives

- Fetch and aggregate F1 news from multiple sources
- Collect race data and statistics using OpenF1 API
- Train AI agents to analyze news sentiment and racing patterns
- Generate betting predictions with initial odds
- Simulate a prediction market for F1 race outcomes

## Data Sources

- **OpenF1 API**: Real-time F1 telemetry, race results, and statistics
- **News Sources**: F1 news articles and updates for sentiment analysis

## Key Features

- Real-time F1 data ingestion
- Multi-agent prediction system
- Betting odds generation
- Prediction market simulation

## Tech Stack

- **APIs**: OpenF1
- **AI/ML**: Agent-based prediction models with Langchain
- **Data Processing**: News fetching and analysis pipeline
